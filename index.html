Hello and Welcome!

I started my programming journey a few years ago while working as an administrator for a regional tutoring company. In that role, I was tasked with managing large amounts of data gathered from student assessments throughout several school districts within the Chicago metropolitan area. My tool of choice was Microsoft Excel. I found that Visual Basic for Applications (VBA), an API built into Excel, enabled me to create efficient custom macros.

Grasping VBA help turn many daunting tasks, that usually took hours to complete manually, into seamless tasks that were completed in completed in just several minutes. Forging an increase in speed utilizing macros and automation was a game changer for me. My interest in programming ingnited and that combined with a desire to create programs for the World Wide Web lead me on my front-end web development journey.
