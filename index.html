Hello and Welcome!

I started my programming journey a few years ago while working as an administrator for a regional tutoring company. In that role, one task was to manage large amounts of data gathered from student assessments within several school districts. My tool of choice was Microsoft Excel. I found that Visual Basic for Applications (VBA), an API built into Excel, enabled me to create efficent custom macros.

Grasping VBA help to turn many daunting tasks, that usually took hours to complete manually, into a seemless task that were completed in under thirty minutes. Forging an increase in speed utilizing macros/applications was a game changer for me. There was no looking back. My interest programming was ingnited combined with an desire to create programs or the World Wide Web.
